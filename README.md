## Run Locally

Clone the project

```bash
  git clone https://github.com/subh-cs/trikon-todo-frontend.git
```

Go to the project directory

```bash
  cd trikon-todo-frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Now, NEXT server is running on `http://localhost:3000`
